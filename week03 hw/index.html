My goals for this course are:

  • Understand the Evolution of the Web
    • Have a comprehensive grasp of the history of the web, from its early stages to its current state.
  • Familiarize with Modern Web Technologies
    • Be well-versed in current web development tools, including HTML5, CSS3, and JavaScript.
  • Develop a Media-Rich, Responsive Website
    • Use a text editor to build a visually engaging website that adjusts seamlessly to different screen sizes and devices.
  • Employ Technical Terminology and Planning Tools
    • Use relevant technical terms and pseudocode to devise solutions, and create site maps and wireframes to plan and structure your project.
  • Utilize JavaScript Libraries and APIs
    • Integrate and apply JavaScript libraries and application programming interfaces (APIs) to enhance web development.
  • Identify Key Aspects of Internet Art and Prominent Artists
    • Recognize the defining features of Internet Art and be able to identify notable creators in this field.
